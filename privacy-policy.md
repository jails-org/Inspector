Jails Inspector Privacy Policy

Jails Inspector is a developer tool for inspecting applications built with the Jails framework.

Data Collection

Jails Inspector does not collect, store, transmit, or share personal information.

The extension operates entirely within your browser and does not send application data or website content to any external server.

Website Content

To provide its functionality, Jails Inspector accesses information exposed by Jails applications running in the current browser tab, such as:

Component hierarchy
Component state
Component relationships
Rendering information
Performance metrics

This information is processed locally and is never transmitted outside the user's browser.

Analytics

Jails Inspector does not use analytics, telemetry, advertising, or tracking technologies.

Third Parties

Jails Inspector does not share information with third parties.

Data Storage

Jails Inspector does not permanently store inspected application data.

Contact

If you have questions about this Privacy Policy, please open an issue in the project's GitHub repository.
