📊 How to use the Inspector

  1. Click the Jails Inspector icon in your browser toolbar
  2. Click "Scan Components" to discover all Jails components
  3. The tree will show parent-child relationships hierarchically
  4. Click on any node to see component details
  5. Use zoom (+/-) and pan (drag) to navigate the tree
  6. Click the reset button (⊙) to return to the default view
